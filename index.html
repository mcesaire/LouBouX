Synchronisation manuelle
L'app se synchronise automatiquement toutes les 30 secondes. Tu peux aussi forcer une synchro.
🔑 Mot de passe de l'application
Protège l'accès à l'app. Choisis le même mot de passe sur les deux téléphones.
Comment ça marche ?
📱
Sur chaque téléphone
Ouvre l'app dans le navigateur et configure le même Gist ID + Token une seule fois.
🔄
Synchro automatique
Toutes les 30 secondes, l'app vérifie si des changements ont été faits sur l'autre téléphone.
✅
Conflit ? Le plus récent gagne
Si les deux téléphones modifient en même temps, c'est la dernière modification enregistrée qui est conservée.
